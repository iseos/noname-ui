# NoName UI

#### Awesome, clean and elegant Visual Studio Code theme.

[NoName UI in marketplace](https://marketplace.visualstudio.com/items?itemName=iseos.noname-ui)

## Notable features

* Dark UI theme.
* Minimalist style.
* Small colour palette.
